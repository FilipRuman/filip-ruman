In this tutorial I will try to give you all of my knowledge that I've gathered
in my journey with NixOS.

This tutorial is designed for people that have very basic nix knowledge. If you
don't have that knowledge I recommend you to watch at least this
[vid](https://www.youtube.com/watch?v=a67Sv4Mbxmc).

If you only want to read about certain things, feel free to skip some pages/
read them in other order.

## Additional Learning Resources

I will be adding links to relevant resources(like relevant GitHub repos) through
out this tutorial, but generally great places to learn about nix/NixOS features
will be:

- [vimjoyter's yt channel](https://www.youtube.com/@vimjoyer)
- [MyNixOS website](https://mynixos.com/home-manager/options/programs)
  especially for not well documented home-manager config. To find options for a
  program in home-manager, you can just go
  [here](https://mynixos.com/home-manager/options/programs) and Ctr+f(program
  name).

- [NixOS wiki](https://nixos.wiki/wiki)
